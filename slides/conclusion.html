Code Review

Test Writing 

"Help me write tests for this model:
- Include happy path
- Edge cases
- Error conditions

[paste your model code]"

Test writing is a great way to practice providing context

Feature Planning 

"Let's plan the implementation of:
- User authentication
- Required features
- Security considerations
- Testing strategy"

Practice breaking down complex features with AI assistance
